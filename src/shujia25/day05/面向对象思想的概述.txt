1、面向对象与面向过程的区别：
    1）面向过程是将我们作为参与者，逐个分析每一步骤的实现，并参与开发
    2）面向对象是基于面向过程的，是将我们从参与者变成了指挥者，不断地创建对象，指挥对象做事情。

2、面向对象的现实举例：
    1）接水：
        面向过程的思想：自己拿起水杯，走到饮水机面前，选择热水还是冷水，接水
        面向对象的思想：祈求同桌帮你接水，同桌拿起水杯，同桌走到饮水机面前，同桌选择热水还是冷水，同桌接水

    2）吃饭：
        面向过程的思想：自己买菜，自己洗菜，自己备菜，自己做菜，自己装盘
        面向对象的思想：点菜，厨师买菜，厨师洗菜，厨师备菜，厨师做菜，厨师装盘

    2）面向对象的思想
        需要考虑几个问题：
            a. 共有几个类？
            b. 类里面都有些什么？
            c. 类与类之间的关系是什么？

        a. 共有几个类？（1、UML进行分析  2、名词提取法）
            3个类： 大象类 冰箱类 测试类

        b. 类里面都有些什么？
            java是一个面向对象的语言，世间万物皆为对象，就可以从属性和行为两个方面区分
            举例：
            桌子：
                属性：材质。价格。大小。。
                行为：放电脑，放鼠标，办公。。。

            题目分析：
                大象类：
                    属性：身高，体重，体型。。。
                    行为：吃，睡，跳进冰箱。。。
                冰箱类：
                    属性： 大小，品牌，价格，颜色。。。
                    行为： 打开冰箱门，关闭冰箱门，制冷。。
                测试类：
                    main方法

        c. 类与类之间的关系是什么？
            测试类中创建大象对象和冰箱对象，实现功能

        面向对象的思想伪代码实现：
            //定义一个大象类
            class Elephant{
                int high;
                int weight;

                public void eat(){...}

                public void jump(){...}
            }

            //定义一个冰箱类
            class IceBox{
                int size;

                public void open(){..}
                public void close(){..}
            }

            //定义一个测试类
            class Demo{
                public static void main(String[] args){
                    //创建一个大象对象
                    Elephant e1 ....
                    //创建一个冰箱对象
                    IceBox i1...
                    //调用冰箱的open方法
                    i1.open();
                    //调用大象对象的jump
                    e1.jump();
                    //调用冰箱对象的close方法
                    i1.close();
                }
            }

4、面向对象思想特点
  1、是一种更符合我们思想习惯的思想
  2、可以将复杂的事情简单化
  3、将我们从执行者变成了指挥者，角色发生了转换

5、面向对象的特征：
    封装
    继承
    多态

6、世间上每一个事物都可以被属性和行为进行描述
    人：                              class Person:
        属性：姓名，年龄，身高                   成员变量：name,age,high
        行为：吃，睡，跑                        成员方法：eat(),sleep(),run()

7、类和对象的关系
    类相当于一张图纸，对象相当于通过图纸造出来的东西，一张图纸（类），可以制造很多辆汽车（对象）
