故事纯属个人虚构，切勿当真，只是为了方便理解多态的向下转型
曹操和曹植是父子关系
class 曹操{
    teach(){教如何打仗的}
}

class 曹植 extends 曹操{
    teach(){教如何下棋的}

    zuoShi(){写诗}
}

突然有一天，父亲曹操外出打仗，其他的国家攻打过来了，曹操不在家，曹植为了让将士们听他的指挥
曹植开始装爹，粘上曹操的假胡子(向上转型)
曹操 c1 = new 曹植();
调用teach()
c1.teach();
过了一会，曹操回来了，这时候，曹植可以做回自己，撕下胡子，脱下父亲的衣服（向下转型）
曹植 c2 = (曹植)c1;
c2.teach();
c2.zuoShi();
